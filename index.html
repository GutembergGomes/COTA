Sobre o Sistema

COTA FRENTE - Sistema de Controle de Moagem e Produção

Versão: 3.2.1

Atualizado em: Dezembro 2024

Este sistema foi projetado para gerenciar e monitorar a produção, permitindo o controle de frentes de trabalho, cotas e eficiência.

Novidades na versão 3.2.1:

  • • CORREÇÃO: Produção por tipo de frente agora usa dados reais ao invés de cota
  • • Distribuição por tipo de frente corrigida
  • • Interface melhorada com animações aprimoradas
  • • Gráficos de distribuição redesenhados
  • • Notificações e alertas melhorados
  • • Performance otimizada
  • • Validações mais robustas
  • • Suporte melhorado para tipos personalizados

Desenvolvido com:

HTML5 CSS3 JavaScript ES6+ TailwindCSS Chart.js

Guia Rápido

  • Ajuste o Planejado (Moagem) e Data/Hora Início para definir o período de análise.
  • Use Gerenciar Tipos para criar tipos de frente personalizados com cores próprias.
  • Configure as Cotas para cada frente de trabalho.
  • Atualize os valores de Realizado e Estoque conforme a produção ocorre.
  • Utilize os Snapshots para registrar estados específicos e acompanhar a evolução.
  • Consulte os Gráficos e Análises para obter insights sobre a produção.
